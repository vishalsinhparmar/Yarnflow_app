# 📦 Install Required Dependencies

Run these commands to install the missing features:

## 1. Install Dependencies

```bash
# Navigate to your project directory
cd c:\Users\Vishal\Desktop\client-projects\YarnFlow_app\Yarnflow_app

# Install AsyncStorage for authentication
npx expo install @react-native-async-storage/async-storage

# Install Toast notifications
npm install react-native-toast-message

# Install File System and Sharing for PDF support
npx expo install expo-file-system expo-sharing

# Install Linear Gradient for better UI
npx expo install expo-linear-gradient
```

## 2. After Installation

Once dependencies are installed, the following features will be available:

✅ **Authentication System**
- Login screen
- Registration screen
- Token management
- Protected routes

✅ **PDF Download & Share**
- Download Sales Challan PDFs
- Share PDFs via native share dialog
- Consolidated SO PDFs

✅ **GRN Approval**
- Approve GRN functionality
- Create inventory lots
- Status updates

✅ **Toast Notifications**
- Better user feedback
- Success/error messages
- Non-intrusive notifications

✅ **Enhanced UI**
- Gradient backgrounds
- Better visual feedback
- Improved styling

## 3. Run the App

```bash
npm start
```

Then press:
- `a` for Android
- `i` for iOS
- `w` for Web (testing only)
