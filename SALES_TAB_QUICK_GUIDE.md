## 🚀 Quick Start

1. **Restart Expo**:
   ```bash
   npx expo start --clear
   ```

2. **Open App** on your phone

3. **Tap Sales Tab** (bottom navigation)

4. **See Real Stats** load automatically

5. **Tap Blue Card** to view all orders

6. **Tap "Create New"** to add an order
